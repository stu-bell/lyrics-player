Choose a folder containing .mp3 files, and .lrc or .txt files for lyrics.

.lrc files with timestamps will play automatically, otherwise use arrow keys.

Choose Folder above to download your playlist file and save it in the same folder named _playlist.tsv

This file is how you tell the app which order to play the tracks, and which audio and lyrics files are related. Open the playlist file in a spreadsheet editor and match the lyrics and audio files. The playlist should contain one row per song, in the sequence you want them to be played. On each row you'll need to provide the exact file names for each lyric file (.txt or .lrc) and, optionally the matching .mp3 file. Watch out for typos in the playlist file! If you get any errors, edit and save your playlist file, refresh the page, and try again. You can include two extra optional columns called 'audio_start' and 'audio_end', which should show the times in seconds the current audio track should start and finish. But if you just want to play the full songs, you can leave these columns blank.

Once you've got all your audio and lyrics files in a folder, and added your playlist file, hit the `Let's Go!` button.
